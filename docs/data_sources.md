# 海況・気象データソース一覧 (Data Source Inventory)

AI予測モデルに使用している各データの格納先、一次ソース、および取得状況のまとめです。

| 項目グループ | 具体的な項目名 | 格納先DBテーブル | **データの一次ソース (Origin)** | 頻度と現状 |
| :--- | :--- | :--- | :--- | :--- |
| **気象・陸上** | 気温、風速、降水量、日照時間 | `weather_history` | **Open-Meteo API** | 日次で充足。 |
| **潮汐** | 潮回り（大潮〜若潮）、潮位 | `tide_history` | **AppSync API (独自システム)** | 日次で充足。 |
| **波浪** | 波高、波向き | `openmeteo_marine_history` | **Open-Meteo API** | 日次で充足。 |
| **河川** | 河川流量（荒川等） | `openmeteo_marine_history` | **Open-Meteo API** | 日次で充足。 |
| **海流フラグ** | 黒潮大蛇行フラグ | (計算ロジック) | 海上保安庁等の発表定義 | 固定値。 |
| **水質 (重要)** | 表層水温 | `tokyo_bay_marine_data` | **千葉県環境生活部 (公式HP)** | **週1〜10日に1回。不足。** |
| **水質 (重要)** | 塩分濃度 | `tokyo_bay_marine_data` | **千葉県環境生活部 (公式HP)** | **週1〜10日に1回。不足。** |
| **水質 (重要)** | 溶存酸素 (DO) | `tokyo_bay_marine_data` | **千葉県環境生活部 (公式HP)** | **週1〜10日に1回。不足。** |
| **水質 (参考)** | 透明度、COD等 | `tokyo_bay_marine_data` | **千葉県環境生活部 (公式HP)** | **週1〜10日に1回。不足。** |

## 今後の改善計画
- **水質データの高頻度化**: 「東京湾環境情報センター（国土交通省）」の連続観測データを取得する新規スクレイピングスクリプトを作成し、`tokyo_bay_marine_data` テーブルを日次データで補強する。
